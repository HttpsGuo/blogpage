# 个人主页

这是一个现代化的个人主页项目，采用响应式设计，支持浅色/深色主题切换，并包含多个交互式功能模块。

## 预览
- 演示地址：[www.httpsguo.cn](https://www.httpsguo.cn)
- 博客地址：[blog.httpsguo.cn](https://blog.httpsguo.cn)

## 功能特点

### 1. 界面设计
- 响应式布局，适配各种设备屏幕
- 支持浅色/深色主题切换
- 现代化的卡片式设计
- 圆角美化和阴影效果
- 平滑的动画过渡效果

### 2. 主要模块
- **个人信息展示**
  - 头像显示（支持旋转动画）
  - 个人简介
  - 备案信息

- **联系方式**
  - 微信（二维码）
  - QQ（二维码）
  - 抖音（二维码）
  - 邮箱
  - Telegram
  - GitHub
  - 悬停提示效果
  - 图标动画效果

- **文章展示**
  - 最新文章列表
  - 热门文章列表
  - 文章标题自动省略
  - 更多文章按钮

- **友情链接**
  - 网站图标悬停效果
  - 按钮式设计
  - 自适应布局

- **装饰元素**
  - 动态时钟
  - 硬币翻转动画
  - 气泡背景效果

### 3. 交互特性
- 按钮悬停效果
- 图标渐显动画
- 二维码弹窗展示
- 文字提示框
- 平滑滚动

## 部署说明
### Nginx 配置
```nginx
server {
    listen 80;
    server_name your_domain.com;
    root /path/to/your/files;

    location / {
        index index.html;
        try_files $uri $uri/ =404;
    }

    location ~* \.(css|js|ico|png|jpg|jpeg|gif|svg)$ {
        expires 7d;
        add_header Cache-Control "public, no-transform";
    }

    # 启用 gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/javascript;
}
```

### 部署步骤
1. 将所有文件上传到服务器指定目录
2. 确保文件权限正确（通常是 644）
3. 配置 Nginx 服务器
4. 重启 Nginx 服务

### 注意事项
1. 确保所有资源路径使用相对路径
2. 检查文件权限和所有权
3. 配置正确的 MIME 类型
4. 启用适当的缓存策略
